# Installation
Contrôle des tondeuses Automower d'Husqvarna via son API.  
  
### Les principes
  
Statut, batterie, erreur, mode, et geofence de la tondeuse  
Démarrer ou Arrêter une tondeuse      
  
### Ajout des périphériques
Cliquez sur "Configuration" / "Ajouter ou supprimer un périphérique" / "Store eedomus" / "Husqvarna Automower" / "Créer"  

  
*Voici les différents champs à renseigner:*

* [Obligatoire] - Login  
* [Obligatoire] - Mot de passe  
* [Obligatoire] - Numéro de tondeuse  

  
Le capteur principal "Status" interroge le cloud Husqvarna. Il restitue les données de la tondeuse numérotée (détail de toutes les tondeuses dans l'xml).     
    
L'actionneur "Commands" permet de contrôler la tondeuse : Start, Stop, Park.
